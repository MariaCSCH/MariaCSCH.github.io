SEMANA 3: Aplicando lo aprendido

Durante la tercera semana se desarrollaron dos prácticas aplicando los fundamentos de HTML, CSS y Tailwind CSS. El propósito fue consolidar los conocimientos sobre estructura de páginas web, maquetación responsive, animaciones, uso de grids, flexbox y trabajo colaborativo con GitHub mediante ramas y merges.

Primera práctica: HTML Y CSS puro

En la Primera práctica, se trabajó exclusivamente con HTML y CSS puro, creando una interfaz con formularios, campos animados y cards con efectos 3D. Se aplicaron conceptos como:
• Propiedades de flexbox para distribuir los elementos.
• Media queries para adaptar el diseño a pantallas pequeñas, medianas y grandes.
• Animaciones con @keyframes y efectos hover para dar interactividad.

Segunda práctica: HTML y Tailwind CSS

Se desarrolló un proyecto colaborativo en GitHub, utilizando Tailwind CSS para construir listas y cards con diseño moderno y adaptable. Se aplicaron grid, flexbox, efectos hover y breakpoints para diferentes tamaños de pantalla. Esta práctica permitió experimentar con un framework actual y comprender la importancia del trabajo en equipo.


Reflexión:

Estas prácticas me permitieron entender la importancia de dominar tanto el CSS tradicional como los frameworks modernos como Tailwind, que agilizan el diseño manteniendo buena estructura. Además, reforcé mis habilidades de colaboración y control de versiones usando GitHub, comprendiendo el flujo de trabajo profesional en equipo.

Referencias:

  • Tailwind CSS. Utility-First Fundamentals. https://tailwindcss.com/docs
