SEMANA 2: HTML/CSS/BOOSTRAP/TAILWIND

Durante la segunda semana se realizaron exposiciones centradas en los fundamentos y el desarrollo práctico con HTML y CSS, así como su integración con frameworks modernos como Bootstrap y Tailwind CSS. El objetivo fue comprender cómo se estructura una página web, cómo se define su estilo visual y cómo se pueden aprovechar herramientas que agilizan el diseño responsive.

Desarrollo teórico

HTML (HyperText Markup Language):
Es el lenguaje base de toda página web. Utiliza etiquetas o “tags” para definir la estructura del contenido, como encabezados, párrafos, imágenes o enlaces.

Código 

<h1>Bienvenido a mi sitio web</h1>
<p>Este es un ejemplo de un párrafo en HTML</p>
<img src="foto.jpg" alt="Ejemplo de imagen">

CSS (Cascading Style Sheets):
Permite dar formato visual al contenido HTML mediante reglas de estilo (colores, tipografía, márgenes, disposición, etc.).

Código 

body {
  background-color: #f3f3f3;
  font-family: Arial, sans-serif;
}

h1 {
  color: #2e86de;
  text-align: center;
}

Bootstrap:
Framework CSS desarrollado por Twitter que permite crear sitios responsive de forma rápida mediante clases predefinidas.
Aquí .btn y .btn-primary aplican automáticamente diseño y color según el tema Bootstrap.

Código 

<button class="btn btn-primary">Enviar</button>

Tailwind CSS:
Framework de utilidades que permite personalizar estilos directamente desde el HTML sin escribir CSS externo.

Código 

<class="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded">
  Guardar
</button>

Cada clase define un aspecto visual (color, tamaño, margen, borde, etc.), lo que acelera el diseño y mejora la productividad.


Reflexión:

Esta semana reforcé la comprensión del frontend y aprendí cómo combinar estructura y estilo para lograr páginas profesionales. Bootstrap y Tailwind me mostraron dos enfoques distintos: uno basado en componentes listos, y otro en personalización total mediante utilidades. Reconocí la importancia de mantener un equilibrio entre simplicidad y eficiencia en el diseño web.
