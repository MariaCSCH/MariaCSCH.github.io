Arquitectura MVC aplicada en JSP / Spring

Jakarta Server Pages (JSP)

Elemento JSP Sintaxis Función
Scriptlet <% código Java %> Lógica Java dentro del JSP
Expresión <%= expresión %> Imprime valores dinámicos
Declaración <%! variables/métodos %> Define variables globales
Directivas <%@ page %> Configura la página JSP

Objetos implícitos JSP

Objeto Uso principal
request Datos enviados por el cliente
response Respuesta HTTP
session Datos de sesión del usuario
out Salida HTML
application Contexto global

Patrón MVC en Java

  • Modelo: Maneja los datos (Entidades, JPA)
  • Vista: JSP
  • Controlador: Servlets / Controllers Spring

Maven

Elemento Descripción
pom.xml Archivo de configuración del proyecto
Dependencias Bibliotecas del proyecto
Repositorio Almacena librerías (mvnrepository)
Comandos mvn clean, mvn install, mvn deploy

Spring Framework y Spring Boot

  • Framework para aplicaciones empresariales Java
  • Implementa Inversión de Control (IoC)
  • Inyección de Dependencias (DI)
  • Spring Boot simplifica configuración y despliegue

Checklist de laboratorio

  • Proyecto creado en start.spring.io
  • Configuración Maven correcta
  • Modelo, Repository y Controller creados
  • CRUD probado con Postman

Reflexión

Esta semana permitió comprender cómo se construyen aplicaciones backend estructuradas usando JSP, MVC, Maven y Spring. La separación de capas y el uso de frameworks simplifican el desarrollo, mantenimiento y escalabilidad de aplicaciones web Java.
