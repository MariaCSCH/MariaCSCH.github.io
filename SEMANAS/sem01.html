SEMANA 1: Introducción al curso

Durante el desarrollo de la primera semana se abordó los fundamentos de la tecnología web, los estándares internacionales (HTML, CSS, JavaScript, HTTP/HTTPS, XML) y el funcionamiento general de frontend y backend. También se explicó la Open Web Platform, el uso del DNS y las herramientas esenciales para el desarrollo web, como Visual Studio Code, GitHub y Figma para diseño UX/UI.
