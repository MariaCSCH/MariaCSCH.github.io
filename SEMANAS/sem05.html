SEMANA 5: Práctica 3/ Introducción a React

En la quinta semana se trabajó en dos actividades principales:
• El desarrollo de la Práctica Calificada 3, una aplicación web dinámica con HTML, CSS y JavaScript puro.
• Una introducción práctica a React, que incluyó la instalación de Node.js y la creación de proyectos en tres entornos distintos (Create React App, Vite y Next.js).
Ambas actividades permitieron consolidar las bases del desarrollo web y avanzar hacia herramientas más modernas para construir interfaces interactivas. El ejercicio consistió en implementar una ruleta dinámica y un sistema de sorteo de equipos de forma totalmente aleatoria. Las funcionalidades clave fueron:
• Ruleta interactiva que gira con clic, tecla Espacio o botón Iniciar.
• TextArea editable, cuyos datos se almacenan y recuperan con LocalStorage.
• Teclas especiales
• Aplicación adicional para generar equipos: los participantes se agrupan de manera aleatoria según número de integrantes o cantidad de equipos.
• Se incluyeron botones para descargar resultados, copiar texto y copiar equipos por columnas.

Link de práctica: https://github.com/ManuelYanceyan/TrabajoN03

Reflexión:

Esta semana me permitió reforzar mi lógica en JavaScript y entender cómo un entorno como Node.js amplía las posibilidades del desarrollo web.
Comprendí que React no es solo una herramienta, sino una forma más estructurada y eficiente de construir interfaces dinámicas.
El proceso de instalación, configuración y prueba de los entornos me ayudó a entender cómo se inicia un proyecto moderno desde cero.
