SEMANA 7: Props y children en react

Esta semana aprendimos cómo los componentes se comunican y comparten información entre sí usando Props y Children, dos de los pilares más importantes de React.
Estos conceptos permiten reutilizar el mismo componente mostrando datos diferentes, y crear estructuras más flexibles dentro de una aplicación.

¿Qué son las Props?

Las Props (propiedades) son datos que se envían de un componente padre a un componente hijo.
Funcionan igual que los parámetros en una función.

¿Qué son los Children?

Los Children permiten incluir contenido dentro de otro componente. Sirven para crear estructuras anidadas y reutilizables.

Reflexión:

Comprendí que las props permiten reutilizar componentes con diferentes valores, mientras que los children facilitan incluir contenido variable dentro de ellos. Ambos conceptos simplifican la estructura y ayudan a construir aplicaciones React más organizadas y potentes.
