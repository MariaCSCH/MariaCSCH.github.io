SEMANA 6: Componentes en react

Durante la sexta semana se desarrolló la teoría de componentes en React, según la guía del Se explicó cómo React renderiza el DOM utilizando ReactDOM.render() y el concepto de DOM virtual, que mejora el rendimiento al actualizar solo las partes necesarias.
Se aprendió a crear componentes de clase y de función, entendiendo que cada uno representa una parte visual o lógica de la aplicación.
También se introdujo el uso de JSX, que permite combinar HTML con JavaScript, e incluir variables, expresiones, condicionales y bucles dentro del código.
Finalmente, se revisaron las formas de aplicar estilos en React:
1. CSS tradicional.
2. Módulos CSS (.module.css).
3. Estilos en línea mediante objetos JavaScript.

Ejercicios de laboratorio

1. Se desarrolló una aplicación en React con siete componentes distribuidos por secciones (encabezado, menú, contenido, formulario, pie, etc.). Se aplicó diseño responsivo con CSS para adaptar la interfaz a distintos dispositivos.

2. Se crearon cuatro componentes anidados, pasando un objeto con datos personales hasta el último componente para mostrarlo en pantalla.

3. Se creó dos componentes en donde el componente hijo pasa el color al componente padre.

4. Se desarrolló 3 componentes un componente padre y dos hijos.

5. Se renderizó los datos de 4 personas y se hizo la presentación en una tabla.

Reflexión:

Esta semana me ayudó a comprender cómo React organiza las interfaces por medio de componentes.
Entendí que cada parte puede construirse de forma independiente, facilitando el mantenimiento y la escalabilidad de una aplicación web.
