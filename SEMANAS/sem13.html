SEMANA 13: Desarrollo Backend con PHP y Laravel

Contenido resumido del documento DesarrolloBackendconPHP.pdf

En esta semana se trabajó el desarrollo backend utilizando el lenguaje PHP junto al framework Laravel. Se inició configurando el entorno local con Apache HTTP Server y PHP, para luego aplicar los fundamentos del lenguaje PHP y finalmente desarrollar una aplicación web usando Laravel bajo el patrón MVC.

Servidor Web Apache

Concepto Descripción
Apache HTTP Server Servidor web que recibe solicitudes HTTP/HTTPS y devuelve contenido web al navegador.
Apache Lounge Distribución de Apache para Windows en formato portable (ZIP).
httpd.conf Archivo principal de configuración del servidor Apache.
Puerto Se configuró el puerto 8080 para el servidor local.

Lenguaje PHP

PHP es un lenguaje de programación del lado del servidor que permite generar contenido web dinámico. El código PHP es procesado por el servidor y el navegador solo recibe el resultado en HTML.

Elemento Uso
Variables Almacenan datos usando el prefijo $
echo / print Muestran información en pantalla
Condicionales if, else, switch
Bucles for, while, foreach
Funciones Permiten reutilizar bloques de código

Composer

Composer es el gestor de dependencias de PHP. Permite instalar y administrar librerías necesarias para los proyectos, incluido Laravel.

  • Instala dependencias por proyecto
  • Utiliza el archivo composer.json
  • Se verifica con composer -V

Framework Laravel

Componente Función
Rutas Definen las URLs de la aplicación
Controladores Manejan la lógica de negocio
Vistas (Blade) Presentación de la interfaz
Modelos (Eloquent) Interacción con la base de datos
Artisan CLI para automatizar tareas

Actividad de Laboratorio

  • Creación de un proyecto Laravel
  • Configuración de conexión a MySQL
  • Creación de modelo y migración Estudiante
  • Uso de controlador y rutas
  • Formulario con Tailwind CSS
  • Registro de datos en base de datos

Checklist

  • Apache funcionando correctamente
  • PHP agregado al PATH
  • Composer instalado
  • Laravel ejecutándose en localhost:8000
  • Formulario registra datos correctamente

Reflexión

Esta semana permitió comprender cómo PHP y Laravel facilitan el desarrollo backend moderno. Laravel organiza el código usando MVC, reduce errores comunes y acelera la creación de aplicaciones web robustas y mantenibles.
