SEMANA 4: JavaScript Basico/Avanzado

Durante esta semana se realizaron exposiciones centradas en JavaScript, el lenguaje de programación que añade comportamiento dinámico a las páginas web. El objetivo fue entender cómo funciona desde su nivel básico hasta las aplicaciones más avanzadas que permiten manipular elementos del sitio y responder a eventos del usuario.

JavaScript Básico

• Sintaxis general: estructura de sentencias, comentarios y punto y coma.
• Variables y tipos de datos: var, let, const; tipos primitivos (string, number, boolean, array).
• Operadores: aritméticos, de comparación y lógicos.
• Condicionales y bucles: control del flujo con if, else, for, while.
• Funciones: bloques reutilizables de código para modularidad.

Código 

function sumar(a, b) {
  return a + b;
}
console.log(sumar(3, 4)); // Resultado: 7

JavaScript Avanzado

• DOM (Document Object Model): permite acceder y modificar elementos HTML desde el script.

Código 

function sumar(a, b) {
  return a + b;
}
console.log(sumar(3, 4)); // Resultado: 7

• Eventos: acciones que el usuario realiza y que el programa puede detectar.

Código 

const btn = document.querySelector("#boton");
btn.addEventListener("click", () => alert("Botón presionado"));

• Funciones flecha: una forma más concisa de escribir funciones.

Código 

const multiplicar = (x, y) => x * y;

• Objetos y arreglos: estructuras que permiten agrupar y manipular datos.
• Promesas y asincronía: ejecución de tareas que requieren esperar (por ejemplo, cargar datos externos).

Reflexión:

Estas prácticas me permitieron entender la importancia de dominar tanto el CSS tradicional como los frameworks modernos como Tailwind, que agilizan el diseño manteniendo buena estructura. Además, reforcé mis habilidades de colaboración y control de versiones usando GitHub, comprendiendo el flujo de trabajo profesional en equipo.
